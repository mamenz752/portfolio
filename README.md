## インストール

プロジェクトをローカルで実行する方法を記述します。

1. 実行に必要なモジュールをインストールする

```bash
$ npm i
```

1. localhost:5173でサーバーを起動

```bash
$ npm run dev
```

1. Storybookを起動する

```bash
$ npm run storybook
```
